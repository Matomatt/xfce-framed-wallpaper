# Live wallpaper for Linux XFCE
A simple script that allows you to play an image sequence on your desktop wallpaper!

__Make sure you have these programs installed:__
- python

## Demonstation

# Usage
__IMPORTANT NOTE:__ Paths must be global, example: `/home/username/Pictures/file` not `./file` <br>

```bash
nohup python3 /fullpath/wallpaper.py /fullpath/Pictures/Wallpapers/ 600 &
```
